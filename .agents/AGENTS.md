# Nakhon Parts Dashboard Workspace Rules

## ⚠️ Markdown File Modification Rule
- **Mandatory Approval Requirement**: Before creating or modifying any Markdown (`.md`) file in this workspace, you **must ask the user for explicit permission first**.
- **Process**:
  1. Prepare the exact content or replacement diff.
  2. Ask the user in the conversation if they approve the modification.
  3. Wait for the user's explicit confirmation before writing or editing the file.
- **Applies to**: `plan.md`, `requirement.md`, `troubleshooting_log.md`, and any other markdown documents in the workspace.
